🚀 QUICK START BuddyMatch

=== SIMPLE LAUNCH (RECOMMENDED) ===
1. Double-click: start-backend.bat
2. Double-click: start-frontend.bat (in new window)
3. Wait 2-3 minutes for both to start

=== MANUAL LAUNCH (Alternative) ===

BACKEND Terminal 1:
1. cd BuddyMatch.Api
2. dotnet run
3. Wait for: "Now listening on: http://localhost:5104"

FRONTEND Terminal 2:
1. cd BuddyMatch.Web
2. npm install
3. npm start
4. Wait for: "Local: http://localhost:3000"

=== ACCESS POINTS ===
- 🌟 Main App: http://localhost:3000
- 🔧 API Test: http://localhost:5104/api/employees
- 📋 Test Page: demo.html (open in browser)
